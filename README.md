# IT Helpdesk Active Directory Lab (AWS + Windows Server)
## Overview 
This project simulates a real-world IT Helpdesk environment using AWS infrastructure, Windows Server, and Active Directory. It demonstrates core IT support skills including user management, access control, troubleshooting workflows, and ticketing system operations.

The goal of this lab is to replicate enterprise-level IT support processes and document them in a structured, professional format.

## Technologies Used

- Amazon Web Services (AWS EC2)
- Windows Server 2022
- Active Directory Domain Services (AD DS)
- PowerShell
- osTicket (Ticketing System)
- Remote Desktop Protocol (RDP)

---

## Lab Architecture

- Cloud-based Windows Server deployed via AWS EC2
- Active Directory Domain Controller configured on Windows Server
- Organizational Units (IT, HR, Sales)
- Role-based security groups
- Simulated end-user environment
- Ticketing system for IT support workflows

## Simulated IT Support Tasks

The following tasks were performed and documented:

- Password resets with enforced password change policies
- Account lock/unlock procedures
- User creation and onboarding workflows
- User disabling for offboarding scenarios
- Group membership management
- Access control simulation

Each task includes screenshots and step-by-step documentation.

---

## Ticketing System (osTicket)

A fully functional ticketing system was implemented to simulate real IT support workflows.

### Example Tickets:
- Password reset request
- Account access issues
- New employee onboarding
- Permission and group access issues

Each ticket includes:
- Issue description
- Troubleshooting steps
- Resolution
- Outcome documentation
